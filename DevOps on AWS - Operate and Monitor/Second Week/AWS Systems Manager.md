As you will see in the next segments of the course, AWS Systems Manager is used as a core component on the Operate steps of the DevOps journey. Systems Manager offers many features that facilitate the management of entire fleets of Amazon Elastic Compute Cloud (Amazon EC2) instances and also automations (runbooks) that can be run in the environment.
